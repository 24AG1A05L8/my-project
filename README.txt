Constitution of India Website

TECH:
Frontend: HTML, CSS, JS
Backend: Node.js, Express
Database: MySQL

HOW TO RUN:
1. Install Node.js & MySQL
2. Import backend/db/laws.sql into MySQL Workbench
3. cd backend
4. npm install express cors body-parser mysql2
5. node server.js
6. Open frontend/index.html using Live Server in VS Code
